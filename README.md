## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd aryan-portfolio
```

2. Install dependencies:
```bash

```

3. Run the development server:

